# 導入

Duo のデフォルトの SDK は buildroot に基づいて構築されており、Duo のファームウェアの生成に使用されます。 SDK には主に次の部分が含まれています。

- u-boot: 2021.10
- linux kernel: 5.10.4
- buildroot: 2021.05
- opensbi: 89182b2
